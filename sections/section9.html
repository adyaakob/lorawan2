5.1 Network Server Solutions

Primary Recommendation: ChirpStack Network Server

  • Open-source LoRaWAN network server
  • Compatible with SG50 gateway and HKT GT-100
  • Device authentication and management
  • Data routing and processing
  • Network monitoring
  • Over-the-air configuration updates

Alternative Option: The Things Network (TTN)

  • Global LoRaWAN coverage
  • Community-driven network server
  • Supports standard LoRaWAN devices
  • Built-in security features
